# Запуск

Для просмотра данных в консоли:
```
python main.py
```
Флаги: 
- `--short` краткий режим для просмостра ответов без подробных расчетов и таблиц
- `--full` режим для просмотра полного решения
- `--task n` запустить только полный вывод определенного номера


Для просмотра данных в интерфейсе:
```
python flet_app.py
```
